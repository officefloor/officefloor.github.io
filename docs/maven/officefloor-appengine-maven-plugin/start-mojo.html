officefloor-appengine:start

Full name:

net.officefloor.maven:officefloor-appengine-maven-plugin:3.39.0:start

Description:

Starts the AppEngine with Datastore for integration testing.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: pre-integration-test.

Required Parameters

Name Type Since Description
<services> List - Path to a yaml file, or a directory containing yaml files, or a directory containing WEB-INF/ web.xml. Defaults to ${project.build.directory}/${project.build.finalName}.
User property is: app.devserver.services.
Alias is: devserver.services.

Optional Parameters

Name Type Since Description
<additionalArguments> List - Environment variables passed to the devappserver process.
User property is: app.devserver.additionalArguments.
Alias is: devserver.additionalArguments.
<automaticRestart> Boolean - Restart instances automatically when files relevant to their module are changed. (default: True)
User property is: app.devserver.automaticRestart.
Alias is: devserver.automaticRestart.
<cloudSdkHome> File - Optional parameter to configure the location of the Google Cloud SDK.
User property is: cloudSdkHome.
<cloudSdkVersion> String - Optional parameter to configure the version of the Google Cloud SDK.
User property is: cloudSdkVersion.
<datastorePort> Integer - Port to run the Datastore on.
User property is: datastorePort.
<defaultGcsBucketName> String - Default Google Cloud Storage bucket name. (default: None)
User property is: app.devserver.defaultGcsBucketName.
Alias is: devserver.defaultGcsBucketName.
<environment> Map - Environment variables passed to the devappserver process.
User property is: app.devserver.environment.
Alias is: devserver.environment.
<host> String - Host name to which application modules should bind. (default: localhost)
User property is: app.devserver.host.
Alias is: devserver.host.
<jvmFlags> List - Additional arguments to pass to the java command when launching an instance of the app. May be specified more than once. Example: "-Xmx1024m -Xms256m" (default: None)
User property is: app.devserver.jvmFlags.
Alias is: devserver.jvmFlags.
<port> Integer - Lowest port to which application modules should bind. (default: 8080)
User property is: app.devserver.port.
Alias is: devserver.port.
<projectId> String - The Google Cloud Platform project name to use for this invocation of the devserver.
User property is: app.devserver.projectId.
Alias is: devserver.projectId.
<serviceAccountKeyFile> File - Optional parameter to configure the key file used for gcloud authentication.
User property is: serviceAccountKeyFile.
<skip> boolean - (no description)
Default value is: false.
<startSuccessTimeout> int - Number of seconds to wait for the server to start. Set to 0 to not wait.
Default value is: 30.
User property is: app.devserver.startSuccessTimeout.
Alias is: devserver.startSuccessTimeout.
<verbosity> String - Optional parameter to configure the gcloud logging verbosity level.
User property is: gcloud.verbosity.
