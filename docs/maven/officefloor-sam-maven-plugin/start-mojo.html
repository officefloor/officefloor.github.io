officefloor-sam:start

Full name:

net.officefloor.maven:officefloor-sam-maven-plugin:3.40.0:start

Description:

Starts SAM for the integration testing.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.

Required Parameters

Name Type Since Description
<samPort> int -

Port for servicing SAM.

Must be specified so that project is clear on integration test ports. This avoids possible changing default port number.


User property is: sam.port.

Optional Parameters

Name Type Since Description
<dockerNetworkName> String - Name of the docker network.
Default value is: officefloor-sam.
User property is: sam.docker.network.
<dynamodbPort> int - Exposed port to connect to DynamoDB.
Default value is: 8001.
User property is: dynamodb.port.
<env> Map - Additional environment properties.
<initiateRequestPath> String - Path to send the initial HTTP request.
Default value is: /.
<isInitiate> boolean - Indicates whether to run initiate HTTP request.
Default value is: true.
<requestTimeout> int - Time out for servicing a request.
Default value is: 60.
